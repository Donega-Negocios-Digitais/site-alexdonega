📌 Resumo da Aula

Leonardo Varricchio apresenta o Visual Prompt Pro, um agente de IA que funciona como engenheiro criativo intermediário entre intenção humana e geração técnica de imagens. A aula demonstra três aplicações práticas principais: (1) Remodelagem de criativos - pegar anúncios existentes de baixa qualidade e transformá-los em peças profissionais mantendo textos originais mas melhorando diagramação, cores e impacto visual; (2) Criação de mockups de produto - gerar múltiplas variações criativas de cenários luxuosos para apresentar produtos em e-commerce mantendo fidelidade absoluta ao item original; (3) Processo iterativo - conceito fundamental de que raramente a primeira geração será perfeita, sendo necessário ciclo de avaliação e refinamento progressivo. Leonardo enfatiza a importância de sempre incluir imagem de referência do produto original, usar “Edit Remix” (não apenas “Remix”) para manter fidelidade máxima, e que o agente pode sugerir ideias criativas além do solicitado inicialmente. O workflow completo inclui expansão de formato usando Adobe Firefly gratuito (2x3 para Stories 1080x1920 ou Feed 1080x1350). O diferencial está na democratização: qualquer pessoa sem conhecimento de design pode criar criativos profissionais conversando naturalmente com a IA e iterando até resultado satisfatório.

🎯 FLUXO DE TRABALHO COMPLETO

ETAPA 1: PREPARAÇÃO INICIAL

O que fazer:

  • Definir objetivo claro (anúncio, mockup, catálogo, remodelagem)
  • Ter imagem de referência do produto/modelo original (se aplicável)
  • Identificar elementos que DEVEM ser mantidos (textos, cores, produto)
  • Listar o que precisa melhorar (diagramação, fundo, iluminação)

Onde executar: Preparação mental/documental antes de abrir ferramentas

ETAPA 2: PRIMEIRA CONVERSA COM VISUAL PROMPT PRO

O que fazer:

  • Enviar imagem de referência (se houver)
  • Descrever intenção em linguagem natural simples
  • Especificar restrições obrigatórias (“manter textos idênticos”, “produto 100% fiel”)
  • Solicitar melhorias desejadas (“melhorar cores”, “fundo mais contextualizado”)
  • Pedir o prompt final: “manda o prompt”

Comandos exemplo:

"Preciso reformular meu criativo. Mantenha TODOS os textos idênticos.
Melhore diagramação, cores e impacto visual. Dá chan 10x. Manda o prompt."
"Quero criar mockups luxuosos para vender esta bolsa na Shopee.
Mantenha produto 100% fiel. Me sugira 5 variações criativas."

Onde executar: ChatGPT com agente Visual Prompt Pro ativo

ETAPA 3: GERAÇÃO NO SORA

O que fazer:

  • Abrir Sora (interface de geração de imagens do ChatGPT)
  • Escolher formato (quadrado, vertical, horizontal)
  • Colar prompt gerado pelo Visual Prompt Pro
  • SEMPRE incluir imagem de referência original
  • Gerar imagem(ns) conforme plano:
    • Gratuito: 1 por vez
    • Plus ($20/mês): 2 por vez + 2 na fila
    • Pro ($200/mês): 4 por vez + 5 na fila

Atalho importante: Clicar em “MyMedia” → último pedido → “Edit Remix” para reabrir prompt com referência original

Onde executar: Sora (ChatGPT)

ETAPA 4: AVALIAÇÃO CRÍTICA

O que fazer:

  • Comparar resultado com objetivo inicial
  • Identificar acertos (o que funcionou bem)
  • Listar problemas específicos:
    • Texto incorreto/mal posicionado
    • Cores fugindo da identidade
    • Produto distorcido
    • Fundo inadequado
    • Falta de “respiro” visual

Critérios de decisão:

  • ✅ Aprovar se: 80%+ satisfatório, pode testar em campanha
  • ⚠️ Iterar se: tem potencial mas precisa ajustes específicos
  • ❌ Recomeçar se: resultado muito distante da intenção

Onde executar: Análise visual no próprio Sora

ETAPA 5: ITERAÇÃO (se necessário)

O que fazer:

  • Print da imagem gerada
  • Voltar ao Visual Prompt Pro
  • Enviar imagem atual + feedback específico:
"Essa imagem ficou quase perfeita. Porém:
- Manter paleta do produto (azul e cinza, não roxo/amarelo)
- Textos separados em blocos com mais respiro
- Alinhamento à esquerda (não centralizado)
- Fundo com academia desfocada
Me gera 2 versões de prompt corrigidas."
  • Copiar novo prompt
  • No Sora: usar “Edit Remix” (não “Remix”) para manter fidelidade à referência original
  • Colar novo prompt + manter imagem original como referência

Regra crítica: Edit Remix > Remix (menos distorções cumulativas)

Onde executar: Visual Prompt Pro → Sora

ETAPA 6: CORREÇÃO DE PRODUTO (se necessário)

Situação: Embalagem/produto saiu diferente do original

O que fazer:

  • Encontrar foto de alta qualidade do produto real
  • No Visual Prompt Pro:
"Este prompt gerou imagem quase perfeita, porém embalagem mudou muito.
Preciso que seja IDÊNTICA à referência.
[Foto 1: como ficou]
[Foto 2: produto original]
Manda prompt corrigido."
  • No Sora: Edit Remix com novo prompt + ambas imagens de referência

Onde executar: Visual Prompt Pro → Sora

ETAPA 7: VARIAÇÕES CRIATIVAS (opcional)

O que fazer quando: Encontrou versão satisfatória e quer testar alternativas

Comando:

"Show! Agora está perfeito. Quero mais 5 variações criativas
mantendo formato e TODOS os textos, porém sugerindo mudanças
que podem aumentar conversão. Seja criativo."

Resultado: 5+ alternativas para teste A/B em gerenciador de anúncios

Onde executar: Visual Prompt Pro → Sora (múltiplas gerações)

ETAPA 8: AJUSTE DE FORMATO (se necessário)

Situação: Imagem gerada vem em 2x3, precisa adaptar para Stories (9:16) ou Feed (4:5)

O que fazer:

  1. Baixar imagem final do Sora
  2. Acessar Adobe Firefly (gratuito): firefly.adobe.com
  3. Fazer upload da imagem
  4. Clicar em “Expandir”
  5. Escolher formato:
    • Stories: 1080 x 1920
    • Feed: 1080 x 1350
  6. Clicar “Gerar”
  7. Avaliar 3-4 variações geradas
  8. Baixar melhor resultado

Limite gratuito: ~25 gerações/mês no Adobe Firefly

Onde executar: Adobe Firefly

ETAPA 9: ADIÇÃO DE TEXTO (se aplicável)

Opção A - Pedir à IA:

"Essa versão ficou linda. Agora adicione de forma luxuosa
uma copy para anúncio no Facebook Ads.
CTA: 'Últimas unidades com 50% OFF'"

Opção B - Editar manualmente:

  • Gerar imagem sem texto
  • Abrir no Canva
  • Adicionar textos, CTAs, selos manualmente

Vantagem da Opção B: Controle total sobre tipografia e posicionamento

Onde executar:

  • Opção A: Visual Prompt Pro → Sora
  • Opção B: Canva

⚙️ TÉCNICAS AVANÇADAS

TÉCNICA 1: MÉTODO CHAN (Amplificação)

O que é: Reforço proposital de instruções usando multiplicadores

Como usar:

  • Adicionar “dá chan 10x” ou “dá chan 100x” no comando
  • IA exagera propositalmente no aspecto solicitado
  • Funciona para: brilho, contraste, impacto visual, fidelidade

Exemplo:

"Tira esse vermelho. Manter paleta azul e cinza. Dá chan 100x!"

Observação: Não é técnica “oficial”, mas funciona empiricamente

TÉCNICA 2: MISTURA DE PROMPTS

O que é: Combinar características de 2+ imagens geradas

Como usar:

"Gostei do fundo branco da Versão 3, mas quero adicionar
o box de texto que ficou na Versão 5. Misture esses 2 prompts."

Resultado: Híbrido otimizado das melhores características

TÉCNICA 3: SOLICITAÇÃO DE SUGESTÕES

O que é: Deixar IA propor ideias criativas além do pedido inicial

Como usar:

"Se tiver alguma sugestão criativa, pode mandar 2 versões."

Resultado: IA adiciona elementos não solicitados que podem aumentar conversão (degradê, texturas, flares de luz, etc.)

TÉCNICA 4: REFERÊNCIA DUPLA

O que é: Enviar 2 imagens como referência simultânea

Quando usar:

  • Pinterest/referência de estilo + produto original
  • Arte existente + novo produto para substituir
  • Modelo desejado + cenário desejado

Como usar:

"Quero arte igual à referência 1 (Pinterest)
porém com paleta de cores da minha marca (azul/preto)
e usando meu produto (referência 2)"

🚨 ERROS COMUNS E SOLUÇÕES

ERRO 1: Produto distorcido na geração

Causa: Não incluiu imagem de referência do produto original
Solução: SEMPRE usar Edit Remix com foto do produto real como referência

ERRO 2: Texto saindo incorreto

Causa: IA tem dificuldade com fidelidade textual
Solução:

  • Reforçar “manter textos 100% idênticos”
  • Gerar múltiplas variações até acertar
  • Última alternativa: adicionar texto manualmente no Canva

ERRO 3: Cores fugindo da identidade

Causa: Comando vago sobre paleta
Solução: Especificar cores exatas: “azul #0066CC e cinza #333333”

ERRO 4: Distorções cumulativas

Causa: Usar “Remix” múltiplas vezes consecutivas
Solução: Sempre usar “Edit Remix” que volta ao prompt/referência original

ERRO 5: Imagem no formato errado

Causa: Sora gera em 2x3 por padrão
Solução: Expandir no Adobe Firefly gratuitamente para Stories/Feed

Passo a passo:

  1. Gerar modelo inicial com características específicas

  4. Variar apenas: ângulo, roupa, cenário, pose
  5. Manter: rosto, proporções, características físicas

Tempo estimado: 30-40 minutos para 5+ cenas
Aplicação: E-commerce de moda, catálogo, Instagram
