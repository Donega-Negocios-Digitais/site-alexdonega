João Adolfo apresenta guia completo para configuração de canais do YouTube, desde criação até otimizações avançadas.

## 📋 Configurações do Canal em Ordem

### **1. Configurações Básicas de Identidade**

1.1 **Definir nome do canal** - Escolher nome estratégico que posicione como autoridade  
1.2 **Configurar identificador** - Criar handle único para busca do canal  
1.3 **Upload da foto de perfil** - Imagem quadrada, centralizada, que represente autoridade

### **2. Configurações de Personalização**

2.1 **Trailer para não inscritos** - Vídeo que apresenta canal para visitantes  
2.2 **Vídeo em destaque para inscritos** - Conteúdo específico para audiência existente  
2.3 **Organizar seções em destaque** - Estruturar como conteúdo aparece no canal: - Shorts na primeira posição - Transmissões ao vivo passadas - Vídeos carregados - Vídeos populares/mais acessados (mover para cima se tiver conteúdo viral) - Vídeos exclusivos para membros - Próximas transmissões ao vivo

### **3. Configurações Visuais**

3.1 **Upload da capa otimizada** - Usar template que funciona em TV, desktop e mobile  
3.2 **Aplicar os 3 pilares na capa**: - O que você faz - Quem você é - Para quem você fala 3.3 **Configurar marca d'água** - Imagem "Inscreva-se" para CTA em todos os vídeos  
3.4 **Definir tempo de exibição** - "Todo o vídeo" para máxima exposição  
3.5 **Publicar alterações** - Salvar configurações visuais

### **4. Configurações Avançadas - Seção Geral**

4.1 **Acessar Configurações** - Menu lateral do YouTube Studio  
4.2 **Selecionar moeda** - Para futura monetização  
4.3 **Definir país de residência** - Brasil ou país correspondente  
4.4 **Inserir palavras-chave do canal** - Termos principais do nicho para SEO  
4.5 **Configurar conteúdo para crianças**: - Marcar "Não" se não for infantil - Selecionar "Quero revisar em cada vídeo" para controle individual

### **5. Configurações Técnicas Avançadas**

5.1 **Vincular Google Ads** (opcional) - Para futuro tráfego pago  
5.2 **Configurar transcrição automática** - Ativar filtro de palavras impróprias se necessário  
5.3 **Configurar anúncios**: - Desativar anúncios baseados em interesses (bloqueia concorrentes, mas reduz RPM) - Deixar ativo se prioriza monetização
5.4 **Permitir compartilhamento de clipes** - Manter marcado  
5.5 **Qualificação de recursos**: - Recursos padrão: ativar - Recursos intermediários: verificar telefone - Recursos avançados: enviar documento de identidade

### **6. Configurações de Permissões e Segurança**

6.1 **Configurar permissões** - Se tiver editores, definir funções específicas sem acesso aos ganhos  
6.2 **Configurações de comunidade**: - Bloquear usuários problemáticos - Ativar redirecionamento em tempo real (para lives) - Bloquear palavras impróprias (criar lista personalizada) - Bloquear links nos comentários (evitar golpes/spam)

### **7. Configurações de Contratos e Monetização**

7.1 **Verificar contratos** - Parcerias de monetização quando canal estiver ativo  
7.2 **Configurar seção de envio** - Informações específicas para uploads

### **8. Finalização**

8.1 **Salvar todas as configurações** - Confirmar mudanças  
8.2 **Verificar funcionamento** - Testar visualização em diferentes dispositivos

**Observações importantes mencionadas por João:**

- Pode levar 1-3 dias para canal aparecer nas buscas
- Template de capa será fornecido na descrição da aula
- Decisão sobre anúncios de concorrentes depende da estratégia (monetização vs proteção)
- Canal dark não precisa mostrar pessoa, pode usar apenas imagens/logos
