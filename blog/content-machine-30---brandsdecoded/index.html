📌 Resumo da Aula

Leonardo Varricchio apresenta o Content Machine 3.0, uma evolução revolucionária que unifica todo o processo de criação de conteúdo em um único super agente GPT. O sistema elimina a necessidade de alternar entre múltiplos agentes, oferecendo um fluxo integrado que vai desde a ideação até a exportação de carrosséis prontos. A aula demonstra o processo completo em 4 módulos práticos: (1) uso do agente GPT para gerar conteúdo estratégico com embasamento cultural, (2) importação de templates no Figma desktop, (3) instalação e configuração do plugin Content Machine 3.0 com suas novas funcionalidades de aplicação automática de textos, troca de imagens e personalização de cores, e (4) criação do carrossel final unindo todas as peças. Leonardo enfatiza que o diferencial não é apenas escrever com IA, mas garantir qualidade através de tese central, movimento cultural conectado e argumento-mãe narrativo. O workflow completo leva apenas 5-10 minutos, transformando artigos, transcrições ou ideias soltas em carrosséis profissionais prontos para publicação.

🎬 VISÃO GERAL DO CONTENT MACHINE 3.0

O Que é o Content Machine 3.0

Super agente GPT integrado que revoluciona a criação de conteúdo ao unificar todo o processo em uma única ferramenta, eliminando a necessidade de copiar e colar entre diferentes agentes.

🤖 MÓDULO 1: USANDO O AGENTE GPT

Senha de Acesso

Senha: blueprint

Duas Formas de Criar Conteúdo

OPÇÃO 1: A Partir de Conteúdo Existente

Quando usar:

  • Artigos completos
  • Transcrições de vídeos/podcasts
  • Pesquisas e estudos
  • PDFs com conteúdo substancioso
  • Material de referência denso

Processo detalhado:

  1. Preparação do conteúdo

    • Para artigos: copiar texto completo
    • Para vídeos YouTube: usar YouTube Transcript
    • Para PDFs: ter arquivo pronto
    • IMPORTANTE: Não enviar links diretos

  2. Iniciar experiência no agente

    • Escolher “Opção 1”
    • Colar texto integral no chat
    • Aguardar análise do agente

  3. Validação de qualidade

    • Agente avalia densidade do conteúdo
    • Pode sugerir voltar se material for fraco
    • Opção de seguir com original ou ajustar

OPÇÃO 2: A Partir de Ideias Soltas

Quando usar:

  • Palavras-chave específicas
  • Insights pontuais
  • Prints de perfil que funcionaram
  • Posts anteriores de sucesso
  • Ideias faladas (usar função ditar!)

Processo criativo:

  1. Input inicial

    • Digite ou dite suas ideias
    • Pode ser uma única palavra-chave
    • Exemplo usado: “marketing digital com AI”

  2. Geração de narrativas

    • Agente cria 5 narrativas criativas
    • Exemplos gerados:
      • “A lógica da automação invisível”
      • “O efeito Fast Content”
      • “A agência o prompt”

  3. Seleção e refinamento

    • Escolher narrativa preferida
    • Possibilidade de pedir mais opções
    • Direcionamento final do conteúdo

Etapas do Processo de Criação

ETAPA 1: Tese Expandida e Análise Estratégica

Componentes do embasamento:

  1. Tese Central

    • Argumento principal da narrativa
    • Ponto de vista único e defensável

  2. Movimento Cultural Conectado

    • Contexto social/cultural relevante
    • Tendências comportamentais atuais

  3. Por Que Isso Prende Atenção Agora

    • Relevância temporal
    • Urgência ou oportunidade

  4. Argumento-Mãe da Narrativa

    • Estrutura principal do discurso
    • Fio condutor da argumentação

Exemplo prático gerado: “A inteligência artificial transformou o marketing digital em uma linha de produção contínua…”

ETAPA 2: Headlines Estratégicas

Importância crucial:

  • Momento de direcionamento final
  • Define tom e ângulo do conteúdo
  • Última chance de impressão pessoal
  • Influencia toda estrutura narrativa

Funcionalidades:

  • Múltiplas opções geradas
  • Possibilidade de pedir mais alternativas
  • Escolha define rumo do copywriting

ETAPA 3: Copywriting dos Slides

Estrutura do output:

  • 10 slides completos
  • Texto otimizado para legibilidade visual
  • Formatação Markdown pronta
  • Divisão clara (Texto 1, Texto 2, etc.)
  • CTA no slide final

ETAPA 4: Seleção de Template

Templates disponíveis atualmente:

  • Template Principal (Twitter/Carrossel)
  • Template 2
  • Template 3
  • Template 4
  • (Lista em expansão contínua)

ETAPA 5: Output Final

Entregáveis:

  • Markdown formatado completo
  • Textos divididos por slides
  • Hashtags sugeridas
  • Agenda de publicação
  • Conteúdo pronto para aplicação no Figma

🎨 MÓDULO 2: CONFIGURANDO O FIGMA

Instalação do Figma Desktop

CRÍTICO: Versão desktop obrigatória para plugin!

Passo a passo:

  1. Acessar: https://figma.com/download
  2. Baixar instalador (Windows ou macOS)
  3. Instalar normalmente
  4. Ignorar: App mobile e instalador de fontes

Observação de Leonardo: “O Figma funciona também numa versão de navegador, mas para a gente conseguir obter todas as funções nele, a gente tem que baixar para o PC, que é programa super leve.”

Importando Templates

Processo de importação:

  1. Localizar botão de importar

    • Ícone de chave/seta
    • Geralmente na tela principal

  2. Navegar até Downloads

    • Selecionar arquivo .fig do template
    • Exemplo: Template-Principal.fig

  3. Aguardar carregamento

    • Template aparece na área de trabalho
    • Pronto para edição

🔌 MÓDULO 3: INSTALANDO O PLUGIN

Download e Extração

Passo a passo crítico:

  1. Baixar arquivo ZIP

    • Disponível na área de materiais da aula
    • Arquivo: Content Machine 3.0.zip

  2. Extrair corretamente

    • Clicar com botão direito
    • “Extrair tudo”
    • Escolher local definitivo

⚠️ AVISO IMPORTANTÍSSIMO: “Uma vez que você escolheu onde deixar os arquivos, deixe eles sempre aqui. Porque se você mexer e deletar essa pasta, por exemplo, mexer ela de posição, botar ela para a área de trabalho, botar ela para uma pasta de documentos, você vai ter que repetir esse processo.”

Instalação no Figma

Processo de instalação:

  1. Abrir template no Figma
  2. Clicar com botão direito
  3. Navegar: Plugins → Development → Import Plugin From Manifest
  4. Selecionar manifest.json
  5. Plugin instalado!

Funcionalidades do Plugin

1. Aplicar Textos

Como funciona:

  • Selecionar todos os frames/slides
  • Colar Markdown do GPT
  • Clicar “Aplicar Textos”
  • Substituição automática instantânea

Melhorias da v3.0:

  • Feedback visual de aplicação
  • Detecção inteligente de frames
  • Mensagem de confirmação

2. Trocar Imagens

Processo automático:

  • Arrastar ou selecionar múltiplas imagens
  • Auto Map: mapeamento automático
  • Possibilidade de reordenar manualmente
  • Aplicação em massa com um clique

3. Alterar Cores

Personalização avançada:

  • Paletas pré-definidas
  • Criação de paletas personalizadas
  • Edição de cores existentes
  • Salvamento local automático

Destaque de Leonardo: “O que você salvar aqui quando você reabrir o plugin vai ficar salvo no seu computador também.”

4. Exportar Carrosséis

Opções de exportação:

  • Via plugin (novo!)
  • Via menu Figma tradicional
  • Formato PNG alta qualidade
  • Escolha de local para salvar

5. Recursos Adicionais

Interface aprimorada:

  • 🌓 Temas claro e escuro
  • 📚 Tutorial integrado
  • 💬 Botão direto para WhatsApp
  • 💾 Auto-save de configurações

🚀 MÓDULO 4: WORKFLOW COMPLETO

Do GPT ao Post Pronto em 5 Minutos

PASSO 1: Gerar Conteúdo no GPT

Checklist de execução:

  • Acessar Content Machine 3.0
  • Escolher método (conteúdo ou ideia)
  • Seguir processo até Markdown final
  • Copiar bloco de texto completo

PASSO 2: Aplicar no Figma

Sequência de aplicação:

  1. Abrir template escolhido
  2. Abrir plugin Content Machine
  3. Selecionar todos os slides
  4. Colar texto (Ctrl+V)
  5. Clicar “Aplicar Textos”

PASSO 3: Ajustes Finos de Texto

Refinamentos recomendados:

Hierarquia visual:

  • Aplicar negrito em palavras-chave
  • Usar variações de peso (Regular/Bold/Black)
  • Destacar conceitos principais

Ajustes técnicos:

  • Tamanho de fonte se necessário
  • Largura das caixas de texto
  • Espaçamento entre linhas

Personalização estratégica:

  • CTA do último slide
  • Créditos/fonte quando aplicável
  • Ajuste do slide de capa

Dica de Leonardo: “Uma coisa que eu gosto muito de fazer é usar algumas fontes aqui em black ou em bold, em negrito.”

PASSO 4: Adicionar Imagens

Fontes recomendadas:

  • Unsplash: Banco gratuito principal
  • Imagens livres de direitos autorais
  • Mínimo 1920x1080 de resolução

Processo otimizado:

  1. Baixar 10 imagens temáticas
  2. Selecionar todas no plugin
  3. Auto Map faz mapeamento
  4. Ajustar manualmente se preferir
  5. Aplicar em massa

PASSO 5: Personalizar Cores (Opcional)

Quando personalizar:

  • Adequar à identidade visual
  • Criar contraste específico
  • Seguir branding da marca

PASSO 6: Revisão Final

Checklist de qualidade:

  • ✅ Textos legíveis e bem posicionados
  • ✅ Imagens alinhadas e em alta qualidade
  • ✅ Cores harmoniosas
  • ✅ CTA claro e atrativo
  • ✅ Créditos/fonte incluídos
  • ✅ Coerência visual geral

PASSO 7: Exportar

Opções de exportação:

  • Plugin: botão “Exportar”
  • Figma: File → Export
  • Formato: PNG alta qualidade
  • Organizar em pasta específica

🆘 TROUBLESHOOTING

Problemas Comuns e Soluções

Plugin não aparece:

  • Verificar se está no Figma Desktop
  • Confirmar extração correta da pasta
  • Reimportar manifest.json
  • Reiniciar Figma

Textos não aplicam:

  • Selecionar frames antes
  • Verificar Markdown completo
  • Tentar colar novamente
  • Confirmar template correto

Imagens não carregam:

  • Verificar formato (PNG/JPG)
  • Testar com menos imagens
  • Reabrir plugin
  • Verificar integridade dos arquivos

Senha sempre solicitada:

  • Normal: uma vez a cada 24h
  • Design intencional do sistema
