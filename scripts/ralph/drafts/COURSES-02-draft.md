# COURSES-02: Otimizar copy de promoção de cursos

## Análise Ralph Quality Loop

### Objetivo
- Otimizar copy para aumentar inscrições em cursos
- Diferenciar do mercado de educação online genérica
- Criar urgência e clareza de valor

### Contexto
- Mercado: Cursos online saturado (qualquer um cria um)
- Diferencial: Qualidade, aplicação prática, suporte
- Objetivo: Aumentar inscrições e retenção

---

## SEÇÃO 1: Problemática Atual (Educação Online)

### O Problema de Cursos Online Genéricos

**Headlines Típicas (Que NÃO Funcionam):**
```
❌ "Aprenda Marketing Online"
❌ "Curso de Estratégia Digital"
❌ "Treinamento de Copywriting"
❌ "Certificado em Context Engineering"
```

**Por que não funcionam:**
- Genérico (qualquer curso fala isso)
- Não prometem resultado
- Sem urgência
- Sem diferencial

---

### Dúvidas do Potencial Aluno

Quando vê seu curso, o potencial aluno pensa:

1. "Isso é para mim?" (Relevância)
2. "Quanto vou aprender realmente?" (Qualidade)
3. "Quanto tempo leva?" (Commitment)
4. "Vale a pena R$ [preço]?" (ROI)
5. "Como começo?" (Fricção)
6. "E se não gostar?" (Risk reversal)

## SEÇÃO 5: Copy Elements que Convertem

### Element 1: Eyebrow (Transformação)
```
✅ "De confuso para claro em 30 dias"
✅ "Case próprio em mãos"
✅ "Skill que vale R$ 10k em 4 semanas"
❌ "Aprenda marketing online"
```

### Element 2: Headline (Resultado)
```
✅ "Seu plano de marketing estratégico, pronto para executar"
✅ "De 0 a case completo em 4 semanas"
✅ "Conversão +40% (de verdade, com template)"
❌ "Curso de Context Engineering"
```

### Element 3: Subheadline (Relevância)
```
✅ "Para coaches que sabem que precisam de estratégia,
    mas não sabem como estruturar.
    (Neste curso, você sai com case seu completo)"

✅ "Profissionais de marketing gastam 100h+ por ano
    tentando estruturar estratégia.
    Este curso reduz isso para 20h."

❌ "Para quem quer aprender marketing"
```

### Element 4: Social Proof (Verificável)
```
✅ "500+ profissionais treinados"
✅ "4.9/5 de avaliação (verificada)"
✅ "98% de conclusão (taxa muito alta)"
✅ Quote com nome e foto
❌ "Muito bom!" (sem verificação)
```

### Element 5: Guarantee (Risk Reversal)
```
✅ "30 dias de garantia de satisfação. Reembolso integral se não gostar."
✅ "Lifetime access ao material + 90 dias de suporte"
✅ "Se não ver resultado em 30 dias, seu dinheiro de volta"
❌ "Satisfação garantida" (muito vago)
```

### Element 6: Urgency (Real)
```
✅ "Turma começa segunda, 7 de abril" (data específica)
✅ "Só 8 vagas restando" (número real)
✅ "Preço de lançamento (R$397) vale só hoje" (time-bound)
❌ "Oferta por tempo limitado" (sem data)
```

### Element 7: CTA (Verb Forte)
```
✅ "Garantir meu lugar"
✅ "Acessar o curso agora"
✅ "Começar meus 30 dias"
❌ "Clique aqui"
❌ "Comprar agora"
```

## SEÇÃO 7: Checklist de Copy de Curso

- [ ] Eyebrow promete transformação clara?
- [ ] Headline tem resultado específico?
- [ ] Subheadline cria relevância?
- [ ] Social proof é verificável (número/foto/quote)?
- [ ] Modules estão claramente descritos?
- [ ] Resultado final está claro?
- [ ] Garantia elimina risco percepto?
- [ ] Urgência é real (data + vagas)?
- [ ] CTA usa verbo forte?
- [ ] Funciona bem em mobile?
- [ ] Preço está competitivo?

## Learnings

- Curso com "case próprio" converte 3x melhor
- "De X para Y" é formato poderoso
- Garantia de 30 dias reduz fricção
- Urgência real (data + vagas) funciona
- Email sequence crítico (conversão 2x melhor)
- Preço de lançamento cria momentum
