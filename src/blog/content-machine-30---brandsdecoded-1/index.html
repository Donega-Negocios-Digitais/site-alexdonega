Content Machine 3.0 - BrandsDecoded 1

🎯 O Que É

O Content Machine 3.0 é um ecossistema completo de criação de conteúdo para Instagram que combina:

  1. Templates validados (estruturas visuais prontas)
  2. Agentes de IA customizados (GPTs especializados)
  3. Plugin automatizado para Figma (aplicação automática de conteúdo)

É um sistema em 3 camadas que trabalham juntas:

┌─────────────────────────────────────────────────┐
│         CAMADA 1: GERAÇÃO DE CONTEÚDO          │
│  (ChatGPT com agentes especializados)          │
│                                                 │
│  • Content Machine 3.0 → Gera ideias e copy    │
│  • Template Master 3.0 → Adapta p/ templates   │
│  • VisualPrompt Pro™ → Cria conceitos visuais  │
└─────────────────────────────────────────────────┘
                     ↓
┌─────────────────────────────────────────────────┐
│      CAMADA 2: ESTRUTURA VISUAL (FIGMA)        │
│  (Templates profissionais pré-configurados)    │
│                                                 │
│  • 1080x1350px (formato Instagram)             │
│  • Layers nomeadas: texto 1, texto 2...        │
│  • Alvos de imagem: imagem1, imagem2...        │
│  • Sistema de cores intercambiável             │
└─────────────────────────────────────────────────┘
                     ↓
┌─────────────────────────────────────────────────┐
│    CAMADA 3: AUTOMAÇÃO (PLUGIN FIGMA)          │

🔧 Como Funciona na Prática

Fluxo de Trabalho Completo

PASSO 1: Gerar Conteúdo no ChatGPT
├─→ Usar agente "Content Machine 3.0"
├─→ Prompt: "Crie carrossel sobre [tema]"
└─→ IA gera: ideias, copy, estrutura

PASSO 2: Adaptar ao Template
├─→ Usar agente "Template Master 3.0"  
├─→ IA adapta quantidade de palavras exata
└─→ Formato: texto 1 - Conteúdo / texto 2 - Mais conteúdo

PASSO 3: Abrir Template no Figma
├─→ Escolher template validado
├─→ Selecionar frames (slides)
└─→ Template já tem layers nomeadas corretamente

PASSO 4: Aplicar com Plugin
├─→ Abrir plugin Content Machine 3.1
├─→ Colar conteúdo adaptado
├─→ Plugin preenche TUDO automaticamente
└─→ 15-30 segundos e está pronto

PASSO 5: Ajustar e Exportar
├─→ Trocar imagens (se necessário)
├─→ Ajustar paleta de cores (1 clique)
├─→ Exportar todos os slides numerados
└─→ Postar no Instagram

🧩 Componentes do Sistema

1. Agentes de IA (GPTs Customizados)

Content Machine 3.0 → Acessar

O que faz:
• Gera ideias de carrosséis do zero
• Cria headlines que chamam atenção
• Escreve copy que engaja e converte
• Adapta tom de voz para diferentes nichos

Como usar:

"Transforme este artigo em carrossel: [link]"

Template Master 3.0 → Acessar

O que faz:
• Adapta conteúdo para estrutura específica de cada template
• Respeita contagem EXATA de palavras por bloco
• Expande ideias quando há pouco texto
• Condensa quando há muito texto
• Mantém 100% da essência original

Como usar:
Enviar conteúdo bruto + especificar template
IA retorna formato pronto: texto 1 - ... / texto 2 - ...

VisualPrompt Pro™

O que faz:
• Cria conceitos visuais para carrosséis
• Gera ideias de imagens e elementos gráficos
• Sugere paletas de cores
• Define hierarquia visual

Como usar:
Descrever conceito → IA sugere execução visual

3. Plugin Content Machine 3.1 (Figma)

O cérebro da automação. Plugin JavaScript que roda dentro do Figma e executa 4 funções principais:

Função 1: Aplicar Textos

INPUT:
texto 1 - Título principal aqui
texto 2 - Subtítulo explicativo
texto 3 - Primeiro parágrafo

PROCESSO:
1. Plugin identifica layers "texto 1", "texto 2", "texto 3"
2. Busca em TODOS os frames selecionados
3. Substitui conteúdo automaticamente
4. Carrega fontes necessárias

OUTPUT:
✅ Todos os textos aplicados em segundos
⚠️ Aviso se algum layer não existir

Tecnologia por trás:

// Regex que reconhece formato
/^texto\s*(\d+)\s*-\s*(.+)$/i

// Busca inteligente de layers
normalizeName("Texto 1") → "texto1"
normalizeName("TEXTO_1") → "texto1"
normalizeName("texto-1") → "texto1"
// Todos são reconhecidos como "texto 1"

Função 2: Aplicar Imagens

INTERFACE DO PLUGIN:
┌────────────────────────────────┐
│ Slide 1: [imagem1] [Escolher] │
│ Slide 2: [imagem1] [Escolher] │
│ Slide 2: [imagem2] [Escolher] │
│ Slide 5: [imagem1] [Escolher] │
└────────────────────────────────┘

PROCESSO:
1. Plugin detecta automaticamente quais slides têm alvos de imagem
2. Você escolhe arquivo para cada alvo
3. Plugin insere imagem no layer correto
4. Modo de preenchimento: FILL (preenche completamente)

INTELIGÊNCIA:
• Se layer "imagem1" for um grupo, busca filho com fills
• Mantém proporção da imagem
• Substitui apenas o fill, preserva bordas/efeitos

Exemplo prático:

Você tem 5 slides:
- Slide 1: tem "imagem1" (foto de capa)
- Slide 2: tem "imagem1" e "imagem2" (2 fotos)
- Slide 3: sem imagens
- Slide 4: sem imagens  
- Slide 5: tem "imagem1" (foto de conclusão)

Plugin mostra interface:
[Slide 1 - imagem1] → você escolhe foto_capa.jpg
[Slide 2 - imagem1] → você escolhe foto_contexto.jpg
[Slide 2 - imagem2] → você escolhe foto_exemplo.jpg
[Slide 5 - imagem1] → você escolhe foto_cta.jpg

Clica "Aplicar" → TUDO inserido automaticamente

Função 3: Aplicar Cores (Sistema Inteligente)

PROBLEMA QUE RESOLVE:
Template usa: Azul, Laranja, Preto
Você quer: Verde, Rosa, Cinza

Manualmente: trocar cor de 50+ elementos um por um
Com plugin: 1 clique

SISTEMA DE SUBSTITUIÇÃO INTELIGENTE:
┌──────────────────────────────────────┐
│ 1. ANÁLISE DE FREQUÊNCIA             │
│    Plugin conta quantas vezes cada   │
│    cor aparece no template           │
│                                      │
│    Exemplo encontrado:               │
│    #FFFFFF (branco) → 45 vezes       │
│    #FF6A00 (laranja) → 23 vezes      │
│    #0F172A (azul) → 12 vezes         │
└──────────────────────────────────────┘
          ↓
┌──────────────────────────────────────┐
│ 2. ORDENAÇÃO POR FREQUÊNCIA          │
│    1ª mais usada: #FFFFFF            │
│    2ª mais usada: #FF6A00            │
│    3ª mais usada: #0F172A            │
└──────────────────────────────────────┘
          ↓
┌──────────────────────────────────────┐
│ 3. MAPEAMENTO INTELIGENTE            │
│    1ª cor antiga → 1ª cor nova       │
│    2ª cor antiga → 2ª cor nova       │
│    3ª cor antiga → 3ª cor nova       │
│                                      │
│    #FFFFFF → #F5F5F5 (sua nova cor 1)│
│    #FF6A00 → #00D4FF (sua nova cor 2)│
│    #0F172A → #1A1A1A (sua nova cor 3)│
└──────────────────────────────────────┘
          ↓
┌──────────────────────────────────────┐
│ 4. SUBSTITUIÇÃO EM MASSA             │
│    Plugin percorre TODOS os nodes    │
│    e aplica o mapa de cores          │
│                                      │
│    Resultado: 80+ elementos trocados│
│    em 2 segundos                     │
└──────────────────────────────────────┘

Paletas pré-definidas:

Padrão:      #F8FAFD, #FF6A00, #0F172A
Corporativo: #F4F5F7, #0052CC, #172B4D
Natureza:    #CEC5AD, #A27B5C, #2C4830
Impacto:     #F5F5F5, #E63946, #1E1E1E

+ Você pode criar suas próprias paletas customizadas

Filtros inteligentes:

Plugin CONSIDERA:
✅ Cores sólidas (não gradientes)
✅ Opacidade > 50%
✅ Fills visíveis
✅ Tipos: Rectangle, Ellipse, Text, Frame, etc.

Plugin IGNORA:
❌ Gradientes
❌ Imagens  
❌ Opacidade ≤ 50%
❌ Fills invisíveis

Função 4: Exportar PNG Numerado

PROBLEMA QUE RESOLVE:
Exportar 11 slides manualmente:
• Selecionar frame
• Exportar
• Renomear arquivo
• Repetir 11 vezes (tedioso)

COM PLUGIN:
1. Selecionar todos os frames
2. Clicar "Exportar PNG"
3. Plugin exporta TUDO numerado automaticamente

NOMENCLATURA AUTOMÁTICA:
01_Capa.png
02_Slide_Introducao.png
03_Slide_Contexto.png
04_Slide_Desenvolvimento.png
...
11_Slide_CTA.png

ORDEM INTELIGENTE:
• Se frames têm números no nome → Ordena numericamente
  Exemplo: "Slide 2", "Slide 10", "Slide 1"
  Resultado: Slide 1, Slide 2, Slide 10 (não 1, 10, 2)

• Se frames não têm números → Ordena por posição espacial
  Exemplo: Frames dispostos em grid
  Lê linha por linha, esquerda para direita

🎨 Diferencial: Sistema de Template Personalizado

Você Pode Criar Seus Próprios Templates

O Content Machine 3.0 não te prende em templates fixos. Você pode:

  1. Criar template do zero no Figma
  2. Gerar prompt customizado para aquele template específico
  3. Usar o mesmo sistema de automação

Processo de criação de template personalizado:

PASSO 1: Criar estrutura visual no Figma
├─→ Definir quantidade de slides
├─→ Criar blocos de texto (texto 1, texto 2...)
├─→ Definir alvos de imagem (se necessário)
└─→ Aplicar seu próprio estilo visual

PASSO 2: Preencher com conteúdo de exemplo
├─→ Escrever textos temporários
├─→ Inserir imagens placeholder
└─→ Verificar que layout funciona

PASSO 3: Gerar prompt personalizado
├─→ Usar Template Master 3.0
├─→ Comando: "Analise esta estrutura e gere prompt"
├─→ IA conta palavras de cada bloco
└─→ IA cria prompt que respeita sua estrutura

PASSO 4: Salvar prompt como comentário no Figma
├─→ Tecla C (criar comentário)
├─→ Colar prompt completo
└─→ Sempre que usar o template, copiar prompt

RESULTADO:
Seu template personalizado funciona EXATAMENTE
como os templates padrão do sistema
