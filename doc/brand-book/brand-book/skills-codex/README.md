# Brand Book Skills Codex

Colecao de skills para orientar decisões e implementacoes técnicas em projetos web, com foco em consistencia, prevencao de bugs e qualidade de entrega.

## Skills Disponiveis

- `skill-orquestrar-brand-book.md` - Orquestra uso combinado das skills e define fluxo ponta a ponta.
- `skill-escolher-stack-web.md` - Escolha e documentação de stack tecnológica.
- `skill-planejar-arquitetura-repo.md` - Arquitetura de repositorio, contratos por pasta e naming.
- `skill-decidir-renderizacao-web.md` - Decisao entre SSG/SSR/Híbrida/SPA/MPA.
- `skill-governar-sistema-cores.md` - Design tokens, temas e mapeamento de cores.
- `skill-governar-tipografia.md` - Hierarquia tipografica, legibilidade e responsividade.
- `skill-governar-icones.md` - Sistema de ícones, acessibilidade e convencoes.
- `skill-estrategia-seo-indices.md` - SEO técnico, metadados, sitemap, robots e monitoracao.
- `skill-operar-cloudflare-dns.md` - DNS/deployment com SSL, cache, troubleshooting e rollback.

## Ordem recomendada para projeto novo

5. `skill-governar-tipografia.md`

7. `skill-estrategia-seo-indices.md`

9. `skill-orquestrar-brand-book.md` (para consolidar entrega)

## Como usar
- Escolha a skill por contexto da tarefa.
- Copie o workflow/checklist da skill e acompanhe o progresso.
- Preencha templates com dados reais do projeto.
- Finalize com checklist de revisão para reduzir regressao.
